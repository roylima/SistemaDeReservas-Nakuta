  • ¿Qué? Un sistema de Salón de Belleza.

Flujo de Eventos del Sistema

A continuación se detallan los estímulos externos e internos que activan los procesos del sistema:

  • El Cliente realiza una solicitud de servicio.
  • El Sistema envía la confirmación al Cliente.
  • El Personal envía una solicitud de registro.
  • El Sistema entrega datos al Personal.
  • El Sistema registra la reserva.
  • El Sistema registra la venta.
  • El Sistema registra los productos.
  • El Sistema registra al personal.
  • El Sistema consulta la información del cliente.
  • El Sistema consulta la información del servicio.

El Diagrama Entidad-Relación (DER) es una herramienta conceptual utilizada en la fase de análisis para representar gráficamente la estructura lógica de la información.

El Modelo Relacional representa la implementación lógica del diagrama E-R.

Esquema de Tablas
  • • Personal (codPersonal, nombre, ap_paterno, ap_materno, zona, nro, calle)
  • • TelefPer (codTelf, codPersonal, telefono)
  • • Cliente (codCliente, nombre, ap_paterno, ap_materno, ci, fecha_nac, telefono, frecuente)
  • • Reserva (codReserva, fecha, hora, codCliente)
  • • Servicio (codServicio, tipo_serv, precio, hora, fecha_servicio, codPersonal, codCliente)
  • • Venta (codVenta, precio_venta, cantidad_prod, fecha_venta, codPersonal)
  • • Producto (codProducto, stock, tipo_prod, nom_prod, fecha_caducidad)
  • • Contiene (codCont, codProducto, codVenta)
  • • Uso (codUso, codProducto, codServicio)
  • • Pide (codPide, codServicio, codProducto, codReserva)
